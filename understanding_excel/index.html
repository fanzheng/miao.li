搜索

上大学的时候，每次上课，我发现我第一节课总是能认真的听，而后面则总是容易犯困，所以，我今天开始讲的是我觉得最有用的，而我觉得最通用的一个就是搜索。

有的时候，别人问我很多问题，我自己也不知道答案，但是我也得回答，我得自己寻找答案，在网上寻找答案，使用搜索。所以，下面就是我的搜索技巧分享。

关于搜索和学习，我将一个一直关注的博客的一篇文章分享一下，作者是李笑来，博客地址http://lixiaolai.com 或者 http://xiaolai.github.io/ ,文章名字是学习学习再学习，通俗点就是，学习怎样学习后再接着学习。读完让人有脱胎换骨的感觉。

使用Google

Google的页面排序比百度好，能得到更相关的结果。这个不是我一个人觉得Google更好，Google索引的页面更多，包括国外的英文网页。不要害怕英文，现在有各种翻译工具，例如http://translate.google.com，有道翻译，百度翻译。谷歌往往能得到更好的结果。谷歌会将最相关的文章放在前面，会将文章按照被其他网站引用的数量排序，也就被大家认可的更多的放在更前面，百度的话，排序方式是，公司给百度推广费更多的放前面，百度自己的网站，如百度文库，百度百科，百度博客，百度知道，百度贴吧，所有百度的网站会放在前面，没错，国内的网站就是这个德行。

使用正确的关键词

我不知道大家是怎样的搜索习惯，但是请使用关键词来搜索，而不是问一个句子问搜索引擎，可能以后搜索引擎能理解你的句子的意思，但是，目前还不行。

提供足够的信息

如果你问的是Excel问题，请尽量提供详细的信息，例如，你是用的Excel 2010还是2013或2003，你用的系统版本，但是，也有的时候不需要提供，根据需要来。你要说明你的需求和问题所在。

搜索－再搜索

网上有几乎所有问题的答案，只要你遇到过的问题，世界几十亿人民可能在你之前就遇到过了，所以答案就在那里，需要你去找到，如果一个关键词找不到，换一个。Google不能找到，换百度找，Excel的问题我一般是先用英文搜索，然后用中文搜，因为我搜的比较多的是VBA的问题，能提供给我好的答案的依次是 Stackoverflow> mrexcel.com>cpearson.com>msdn.microsoft.com

有意识的提高自己获取信息的能力

才最基本的我们每天需要使用的Outlook查找邮件的办法，例如你搜索益阳海大,如果你输入益阳海大，那么就有含益阳也有含有海大的邮件，但是如果输入"益阳海大",那么就只会有含有益阳海大四个字的邮件。或者我现在几乎尽量全部使用印象笔记来做笔记，因为，我可以搜索里面全部的信息。我可以使用手机，我自己的电脑，或者世界上任意一台联网的电脑查看修改我的笔记。

怎样使用宏

什么是宏

什么是宏？查了一下午百度百科

计算机中宏又称为宏命令，即通过特殊的控制语，将一系列的动作简便话，即，一种批处理的程序。

宏的英文是Macro，而我们经常看到的VBA是什么意思呢？VBA即Visual Basic for Application，Visual BASIC1是一种编程语言，或者更准确的说BASIC是一种编程语言。这里Application是指的Office套件。所以VBA是针对Office程序的一种编程语言，那么VBA和宏它们是什么关系呢？

宏和VBA

宏是使用VBA实现的。宏是批处理程序，而微软选择了使用BASIC来实现宏，然后把这个版本的BASIC叫做VBA。打个比喻，宏是衣服，而VBA是棉线。

那么宏这个衣服难不难做呢？还好，因为又宏的录制工具，我们不需要知道怎样种植棉花，收割，然后将棉花做成棉布，我们可以直接用棉布来缝制衣服。 下面说明怎么来缝补制作衣服。

录制宏

有三个方式进入录制宏的界面，最快捷的方法，状态栏左下角的录制按钮，打开后可以看到这个界面，解释一下。 当前宏工作簿是指当前打开的工作簿，这个宏只能在当前的工作簿中使用，个人宏工作簿，则是一个隐藏的工作簿，如果有这个工作簿，每次在Excel打开的时候Excel都会打开它，这样的话，你录制的宏可以适用于所有的工作簿。

设置快捷键，支持Ctrl 加字母和Ctrl ＋ Shift ＋ 字母的组合，因为Ctrl＋字母的组合，Excel差不多已经占有完了，所以推荐使用Ctrl ＋ Shift ＋字母的组合。（Shift＋字母也就是字母的大写形式）。

也可以在下面说明这个宏的用途或者使用方法，这样你以后再来看的时候，就能想起来这个宏的用途。 点确定开始录制。 我在这里录制两个宏

  1. 将这个工作表中的所有”求和项:”这几个字符替换掉

  2. 设置选定的区域的一个特殊的单元格样式。 #### 替换字符宏录制 当你看到宏录制按钮变成蓝色的正方形的时候，那么宏正在录制，录制是指，Excel正在记录我们的动作，当我们需要的时候可以用宏来帮我们重复这些动作。 我要做的动作：

  3. 按Ctrl ＋ H打开替换窗口
  4. 输入”求和项:”这几个字符。
  5. 点全部替换

这个宏就录制完成了，点刚刚那个蓝色的正方形停止录制。 那么怎么使用我们刚刚使用的这个宏呢？ 在视图标签，有个宏按钮，点击查看宏，在这个对话框中，我们可以看到刚刚录制的宏，点击运行就可以运行这个宏。 这个宏没有什么实际的用途，想像有一天，你的领导发给你很多表，要你一个个打开，然后全部做同一个修改，你录制一个修改的宏，然后你需要做的事情就是，打开工作表，运行宏。这个能一定程度上减少工作量。

设置单元格格式宏

前面说明了宏怎么用，我们知道宏是用VBA实现，那么录制宏的时候，Excel是怎样用VBA来记录的呢？ 我们可以按ALT ＋ F11打开VBA编辑窗口（Visual BASIC Editor，简称VBE）。将Excel和VBA窗口上下放，在Excel中开始录制一个宏，先不做什么，记住这个宏的名称，在VBA窗口编辑器中找到对应的这个宏，然后，我们在Excel中开始操作，选择一个区域，例如”A1:B2”，注意VBA编辑窗口的内容。然后右击，选择单元格格式，设置一个我们想要的单元格格式，对单元格样式进行自定义，确定。注意VBA窗口的变化。回到Excel窗口，停止录制。我们来看VBA窗口的代码。我一句句解释下。'号后面的表示是注释。

sub setformat() 'Sub表示 sub routine，就是子过程的意思。
	Range("A1:B2").Select  '选择A1:B2
	Selection.NumberFormatLocal = "#,##0_);[红色](#,##0);_(* ""-""??_)"
	'设置单元格样式。
end sub '表示Sub routine的结束

每一个宏都是用这样的过程来记录的，我们可以使用VBA编辑器来对 它进行修改，这个宏只有两句话，第一句话就是选择单元格，我想，如果我把选择这句删掉，以后我每次运行它时候我先选择的需要设置单元格，这样这个宏就是设置我选择的区域的单元格样式。或者，在开始录制宏的时候，不把选择的区域的动作录下来。

我写VBA的时候呢，很多动作并不知道，所以，我先把这个动作录制一下，看一下其中的代码，然后对其进行修改，所以，写VBA其实是很简单的。

Excel 用户界面

Ribbon Design

从Office 2007开始，微软开始使用Ribbon Design的用户界面，Ribbon是使用不同的标签，将不同类型的命令类型分类，刚刚开始使用的时候会感觉到不方便，用久了之后，就会非常习惯。

详细介绍Ribbon

Ribbon上的按钮是按用户的常用性来分类的，所有的常用的按钮放在开始标签里面了，然后后面的是根据功能性来分类的，例如插入，用来处理各种Excel的组件，图片，透视表，图表等等。数据，用来处理大量相关的 其实有点像数据库相关的东西，进行大量操作的时候相关的工具，公式是所有与公式相关的工具，将所有公式分类列出来，如果我前面说过数据类型，那么也就有按数据类型分类的，例如文本的数字的。这里是按用途来分量。统计的，数学的，文本的，财务的。还有与公式相关的工具，比较常用的主要是几个。后面公式和函数会详细讲解。

双击标签名称或者右上角的^来隐藏标签下面的工具按钮，这样也就有更多的可视区域。

快速访问栏 (Quick Access Bar)

可以将在工具栏各个标签的工具添加到快速访问工具栏，实现快速访问，快速找到那个功能，不需要一个个标签切换去换。也可以将自己的宏添加到快速访问栏，快速访问栏也可以放到工具栏下面。

工具栏 （ Ribbon）

这个就是Ribbon界面，按不同的用途将工具按钮分类。这里已经有大量的工具，但是，Excel远远远远不止这么多的工具，有大量的不常用的工具隐藏起来，可能有的时候需要这些工具 同样的，可以在自定义工具栏中找到他们。

名称和公式栏

这个工具在审阅和公式使用的时候非常有用。名称可以看到当前选择单元格的位置，也可以在这里输入地址，快速跳到输入的区域。下拉可以看到一已经定义的区域的名称。公式栏可以方便看到整个长的公式。

数据区域

具体不知道这个区域的官方名称，所有数据都在这里数据输入，图表也在这里插入。左下角是工作表导航工具，<，>是前一个或者下一个工作表，|<,>|是跳转到第一个或者最后一个工作表。小技巧，右击>或者<,选择更多工作表，可以看到选择所有工作表以快速激活某个工作表。(原谅我截图是使用的Mac版的Excel)

可能一个两个工作表的时候这样的好处不大，当你有几十个工作表的时候，效果非常明显。

状态栏 （Status Bar）

最下方这条，一般显示的是重要的Excel信息。像所有其他的Excel组件一样，可以自己定义，左边是工具，如宏录制按钮，右边是工具类。选定区域中的求和，最大值。

公式和函数

Word中也可以插入表格，但是我们选择使用Excel因为它可以使用公式和函数，公式和函数让Excel成为Excel。

理解公式和函数

公式是formula，函数是function。公式是以＝开始，使用操作符。函数是接受0个或者多个参数，返回值的。这句话真的比较难看，不知道怎么造这个句好。

公式

操作符就是我们刚刚开始学数学的时候使用的+,-,*,/2，当然在Excel中还有&。公式是使用操作符对数字，字符进行操作。例如

= 1 + 1
= 2 - 1
= 2*2
= "A1"&"B2"

小学内容，就不用解释了，所以公式是不需要学习的，需要学习的只是函数。

函数

函数一般就是一个英文单词，然后一个括号，有时候括号里面需要输入一些东西，我们需要知道的是

  1. 我们需要某个功能的时候找到正确的函数
  2. 函数括号里面输入什么。

对于问题1，在Excel的右上角，有个问号图标，点击它是打开Excel的帮助文件，再进去点击函数参考，可以看到按用途分好的所有的Excel函数。

函数需要参数，（也有不需要的），那么什么是参数？就像公式，如果你有=1+的话，还是得需要在+号后面还有一个数字。函数同样需要有可以操作的对象。

= 1 + 2
= sum(1,2)

这两个都是等于3.

怎么理解参数的话，写一个自定义函数理解一下，我们写一个计算圆面积的公式。

Function Circlearea(r As Double) '需要一个参数，r，圆的半径
	Circlearea = r * r * 3.14159236 ' 半径*半径*pi
End Function

将上面代码粘贴到VBA编辑器窗口的Personal.xlsb文件的某个模块中，然后在输入公式的时候，点公式栏的Fx按钮，选择自定义函数。

上面这个就是一个最简单的函数，Excel的自带的函数也是同样的道理，是根据你的参数，也就是上面r,半径。

=VLOOKUP(lookup_value,table_array,col_index_num,[range_lookup])

上面是vlookup的各个参数，v for Vertical,也就垂直的查找，至上而下的查找，look up就是查找的意思，同理，hlookup，h for Horizontal, 就是横向的水平的查找。很好记忆，如果你记得这两个单词。一个个的解释的这四个参数。lookup_value 要查找的值，lookup是查找的意思，value是值的意思，table_arrary 表格序列，也就是就要查找的数据区域，col_index_numfor column index number,也就是列的索引号。[range_lookup]这个参数的名称意义不明晰，[]表示这个参数是可选的，可以输，也可以不输，不输的话，Excel会使用默认值。但是在实际的使用过程中，会提示你选择True 近似查找或者 False 精确查找，你也知道这个参数怎么输。
