Excel只选择可视区域

April 26, 2014

昨天需要加工一个数据，是一些数据的单位有问题，大部分的数据是用的单位吨，而一部分数据使用的是单位吨，所以有1000倍的差距，需要手工修改，是的，使用Excel的选择性粘贴实现，方法是

  1. 筛选需要操作的数据
  2. 在一个空白位置输入1000
  3. 选择需要修改的区域，然后右击选择选择性粘贴
  4. 在选择性粘贴的对话框中选择除

但是，仅仅有一个问题，操作之后呢发现筛选出去的数据也除了1000，我第一次操作没有意识到有这个问题，所以，做出来的数据又全部有问题。这个问题怎么解决呢？方法就是 ALT+;键。 这个快捷键会让Excel只选中可视区域，筛选的区域或者隐藏的区域不会被选择，所以这样操作的时候不会有问题。
