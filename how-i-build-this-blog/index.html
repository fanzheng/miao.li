这个博客怎样建立的

March 30, 2014

这个博客是使用的Jekyll建立的，放在Github上。因为对git命令不是很熟，所以还是用的SourceTree来提交更新。

具体怎样安装jekyll

因为我是用的OS X，系统自带了Ruby，所以，使用

sudo gem install jekyll

就可以安装jekyll了

jekyll new blog

新建一个文件夹，在blog文件夹中放置你的博客。 在网上下载主题。

jekyll serve

在本地创建和预览你的博客。满意后，将博客传到如github pages上。

编辑博客的话，在Mac上使用的是Mou这个编辑器，在iPad上使用editorial这个编辑器，在手机上使用draft。使用Dropbox同步，在Mac上#使用jekyll build后用source tree更新到GitHub Page上。
